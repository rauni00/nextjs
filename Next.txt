Next.js is a flexible react framework to build web application 

-> DOM
The DOM is an object representation of the HTML elements. 
It acts as a bridge between your code and the user interface,
and has a tree-like structure with parent and child relationships.

  

Next.js provides features for both the development and production stages of an application

Next.js handles much of these code transformations and underlying infrastructure to make it easier for your 
application to go to production.

This is made possible because Next.js has a compiler written in Rust, 




With Next.js, three types of rendering methods are available: 
Server-Side Rendering, 
Static Site Generation, 
and Client-Side Rendering.
